Lilia

A Scheme-like programming language running in the browser.

What is Lilia?

Lilia is a Scheme-like programming language with an interpreter created to be easily accessible. You can access the interpreter from everywhere using a modern web browser.

What is the difference with the other online Scheme interpreters?

Maybe you have already used the online interpreters of repl.it or even قلب (pronounced 'alb). They are quite great to check whether some simple code works. What Lilia tries to do (or plans to do) is to provide the programmer with more user-friendly messages and helpful tools. You can try the interpreter of Lilia here.
